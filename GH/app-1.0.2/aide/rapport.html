Rapport

Les rapports qui seront générés par GH seront enregistrés dans le dossier qui sera au préalable spécifié dans les paramètres sous Dossier HTML. Les rapports une fois générés seront automatiquement consultables dans le navigateur par défaut.

Une fois les rapports générés, ils sont aussi consultables sans l'utilisation de GH. Il s'agit d'aller dans le dossier de votre ordinateur que vous avez spécifié dans les paramètres sous Dossier HTML.

Dans ce dossier, vous ouvrez le dossier qui a le même nom que le nom de votre fichier GEDCOM. Si vous avez dans ce dossier le fichier index.html, ouvrez-le.

Pour consulter un ficher en particulier, allez au choix dans le dossier individus ou familles. Le nom du fichier aura pour nom l'identité (ID) de l'individu ou de la famille.

Individu

Le rapport d'un individu est généré avec un clique simple sur la ligne d'un individu de la liste Individu et en cliquant sur le bouton Individu. Aussi on peut doublecliquer sur la ligne d'un individu de la liste Individu. Une fois le rapport généré, il sera ouvert automatiquement dans le navigateur par défaut.

Famille

Les rapports qui seront générés par GH seront enregistrés dans le dossier qui sera au préalable spécifié dans les paramètres sous Dossier HTML. Les rapports une fois générés seront automatiquement consultables dans le navigateur par défaut. Le rapport d'une famille est généré avec un clique simple sur la ligne d'une famille de la liste Famille et en cliquant sur le bouton Famille. Aussi on peut doublecliquer sur la ligne d'une famille de la liste Famille. Une fois le rapport généré, il sera ouvert automatiquement dans le navigateur par défaut.

Les rapports suivants seront générés.

Attention

Balise RESN

La balise RESN, avis de restriction (restriction notice), est utilisée dans la version 5.5 de GEDCOM pour les données d'individu seulement, dans la version 5.5.1 de GEDCOM pour les données d'individus et de familles, dans la version 5.5.5 de GEDCOM n'est pas utilisée.

Cette balise si utiliser (cette balise est optionnelle) indique que l'information est.

● Confidentiel
● Verrouillé
● Privé

Si l'un de ces indicateurs n'est pas affiché, cela ne veut pas dire que l'on peut librement distribuer les rapports. Vous devez avant distributions de vous assurer que les informations contenues dans les rapports ne vont pas à l'encontre des lois de juridiction qui s'applique à vous.

Média

Les médias sont assujettis aux lois de juridiction qui s'applique à vous. Ils sont aussi assujettis aux conditions d'utilisation de certaines de vos sources. Il est important, si vous distribuez vos rapports, de ne pas les inclure dans vos rapports si ces restrictions s'appliquent.

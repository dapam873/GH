Introduction

Ce logiciel permet d'extraire les informations d'un fichier GEDCOM sous forme de fichier HTML pour tous les individus et toutes les familles.

Pour consulter une fiche, une fois qu'un fichier GEDCOM est lu, soit dans la liste individu ou la liste famille, on choisit l'individu ou la famille et on l'affiche dans son navigateur par défaut.

On peut aussi générer toutes les fiches d'individus et de famille classer dans une page indexée par ordre alphabétique et des familles indexées par ordre alphabétique de conjoint et de conjointe.

Une fois toutes les fiches générées, on peut aussi consulter les fiches sans le logiciel en allant dans le dossier des fiches HTML, sélectionner le dossier portant le même nom que votre fichier GEDCOM et doublecliquer le fichier index.html.
